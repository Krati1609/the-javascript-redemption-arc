# the-javascript-redemption-arc

making friends with the language i blamed for everything.

I've spent a good amount of time avoiding javascript because it always felt a little... confusing. 
The weird syntax, the unexpected behaviour, and the classic "wait, why does this work?" moments definitely didn't help :)

So this repo is me finally trying to understand javascript instead of just blaming it.
Learning concepts, building tiny projects, breaking things, fixing them, and occasionally wondering why my code worked five minutes ago but doesn't anymore.
Let's see how this redemption arc goes.

let's see if the enemy becomes an ally.
